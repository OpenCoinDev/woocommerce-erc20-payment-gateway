## 安装

> 在安装本插件前，需要安装 WooCommerce

1. 下载插件
2. 安装插件

## 配置

1. 打开 **WooCommerce** -  **设置** — **付款**.
2. 启用 **使用 ERC20 Token 支付**
3. 设置钱包地址、ABI、收款地址等信息即可。
