## Install

> You need Install WooCommerce at first.

1. Download Plugin
2. Install Plugin

## Configure

1. Open **WooCommerce** -  **Settings** — **Payments**.
2. enable **Pay with ERC20 Token**
3. set Wallet Address、 ABI、Contact Address
